# Nalakath Holdings Ledger

Premium iOS-style Accounting & ERP system for multi-division holdings.

## Getting Started

This is a Next.js 15 application using:
- **Firebase Auth & Firestore**: For secure data and identity management.
- **Genkit**: For AI-driven financial insights.
- **ShadCN UI & Tailwind**: For a premium iOS-inspired design.

## Local Development

```bash
npm run dev
```

## Deployment

### Vercel (Recommended)
Refer to [VERCEL_GUIDE.md](./VERCEL_GUIDE.md) for step-by-step instructions.

### Firebase App Hosting
The project is already configured with `apphosting.yaml` for Google Cloud deployments.
