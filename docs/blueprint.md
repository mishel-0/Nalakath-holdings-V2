# **App Name**: Apex Ledger

## Core Features:

- Multi-Division & Company Management: Create, manage, and switch between multiple companies or divisions with separate ledgers and consolidated financial views for Nalakath Holdings structure.
- Double-Entry Accounting Core: Implement a full double-entry accounting system including Chart of Accounts, Journal Entries, General Ledger, and Trial Balance generation.
- Project Cost & Profitability Tracking: Track material, labor, and other expenses against project budgets, compare actuals vs. budget, and determine profitability per construction/business project.
- Invoicing & Expense Transaction Management: Create sales/purchase invoices, track payments, categorize expenses, and manage customer/vendor details across divisions.
- Core Financial Reporting: Generate fundamental financial statements including Profit & Loss, Balance Sheet, and project-specific reports with export options for multiple entities.
- Real-time Dashboard & Secure Access: Provide a real-time financial overview through an intuitive dashboard and implement robust role-based access control (Admin, Accountant, Manager, Viewer).
- AI Financial Insights & Anomaly Detection Tool: Leverage an AI tool to detect financial anomalies, identify unusual transactions, predict cash flow issues, and suggest cost optimization strategies based on historical data.
- Asset Management: Track company assets, calculate depreciation, and manage asset lifecycle.
- Loan & Liability Management: Track loans and liabilities, including EMI tracking, interest calculation, payment schedules, and outstanding balances.

## Style Guidelines:

- Primary color (for interactive elements and highlights): A professional, balanced blue ('#4D7699') evoking trust and reliability. This choice complements a clean, minimal design approach.
- Background color: A very light, slightly cool-toned off-white ('#F6F7F9') providing a pristine canvas, maintaining harmony with the primary color while ensuring excellent legibility for light mode.
- Accent color (for subtle secondary highlights and interactive elements): A vibrant yet harmonious sky blue ('#5AC2E8') offers a gentle contrast, drawing attention without being overly bold, consistent with a clean aesthetic. The palette is designed for a light scheme, adapting contrasts for dark mode.
- Headline and body font: 'Inter' (grotesque-style sans-serif) for its modern, neutral, and highly legible qualities, ensuring a clean hierarchy consistent with Apple design principles across all text.
- Utilize minimal, elegant vector icons, preferably monochrome or dual-tone, that align with the SF Symbols aesthetic to maintain a native iOS-like user experience.
- Employ responsive design for a web app with a mobile-first approach, featuring an iOS-style bottom navigation for mobile and a sidebar navigation for desktop/tablet. Implement floating cards, modular sections, and sticky headers with blur effects to reinforce glassmorphism.
- Cards and interactive elements will feature generous rounded corners, with radii ranging from 16px to 24px, providing a soft and inviting touch to the UI.
- Implement smooth 'ease-in-out' animations and transitions, micro-interactions for hover/tap feedback, subtle parallax effects, and skeleton loading states to enhance perceived performance and fluidity, matching a premium user experience.
- All interactive elements, cards, and UI layers will integrate glassmorphism through frosted glass effects, background blur, soft shadows, and subtle gradients, creating depth and a sophisticated visual aesthetic.
